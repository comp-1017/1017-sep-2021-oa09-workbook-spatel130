Create navigational links within a web site

There are serveral techniques to navigating within a web site. The semantic tag for navigation is the <nav> tag. You can navigate through your web site by using relative addressing. For our examples we will use the unordered list tag (<ul>). Our example will consist of:

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)
  • going to a specific place on another page

Navigational menus are placed in your <header semantic area. They may also be placed in your <footer> area.
